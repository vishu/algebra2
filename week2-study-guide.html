Rational Exponents

You already know integer exponents. Rational exponents extend that idea to fractions — and they connect directly to radicals. The key is understanding what the notation means, not just memorizing rules.

Definition

For any real number a and integers m and n (n > 0, and a ≥ 0 when n is even):

a^(1/n) = the nth root of a = ⁿ√a

a^(m/n) = (ⁿ√a)^m = ⁿ√(a^m)

Both forms are equivalent — take the root first or the power first, same answer.

Why this definition makes sense

If exponent rules are going to keep working, then a^(1/2) · a^(1/2) must equal a^1 = a. So a^(1/2) must be the number that multiplied by itself gives a — which is exactly √a. The definition isn't arbitrary; it's forced by consistency.

Examples 8^(1/3) = ∛8 = 2 16^(3/4) = (⁴√16)³ = 2³ = 8 27^(-2/3) = 1/27^(2/3) = 1/(∛27)² = 1/9 (-8)^(1/3) = ∛(-8) = -2 (odd root of negative: OK) (-4)^(1/2) = √(-4) = not real (even root of negative: not real)
Negative bases

Even roots of negative numbers are not real — they lead to complex numbers (covered in section 4). Odd roots of negatives are fine and give a negative result. This distinction matters when simplifying.

All the exponent rules still apply a^(m/n) · a^(p/q) = a^(m/n + p/q) (add exponents, same base) (a^(m/n))^(p/q) = a^(mp/nq) (multiply exponents) (ab)^(m/n) = a^(m/n) · b^(m/n) (distribute over product) a^(-m/n) = 1 / a^(m/n) (negative → reciprocal)
Worked Example — Simplifying with rational exponents
Simplify: (8x⁶)^(2/3)
(8x⁶)^(2/3) = 8^(2/3) · (x⁶)^(2/3) distribute exponent 8^(2/3) = (∛8)² = 2² = 4 (x⁶)^(2/3) = x^(6 · 2/3) = x⁴ Answer: 4x⁴ // The key move: distribute the rational exponent over the product, then handle each piece.
Practice — Rational Exponents
  1. Rewrite as a radical: x^(3/5)
  2. Rewrite with a rational exponent: ⁴√(x³)
  3. Simplify: 32^(3/5)
  4. Simplify: (27a⁹)^(2/3)
  5. Simplify: x^(1/2) · x^(2/3) (hint: add the exponents, find common denominator)

Simplifying Radical Expressions

A radical is in simplified form when: no perfect nth powers remain under the radical, no fractions are under the radical, and no radicals are in a denominator. Simplifying is mostly about pulling perfect powers out from under the root sign.

Product and Quotient Rules for Radicals

ⁿ√(ab) = ⁿ√a · ⁿ√b      (only when both sides are defined)

ⁿ√(a/b) = ⁿ√a / ⁿ√b      (b ≠ 0)

These let you split a radical apart to isolate perfect-power factors, or combine separate radicals into one.

Worked Example — Simplifying a square root
Simplify: √72
Find the largest perfect square factor of 72: 72 = 36 · 2 √72 = √(36 · 2) = √36 · √2 = 6√2 // Strategy: factor the radicand, pull out perfect squares.
Worked Example — Simplifying with variables
Simplify: √(50x⁵y⁴) (assume x, y ≥ 0)
Break into perfect-square pieces: 50x⁵y⁴ = 25 · 2 · x⁴ · x · y⁴ √(50x⁵y⁴) = √25 · √(x⁴) · √(y⁴) · √(2x) = 5 · x² · y² · √(2x) = 5x²y²√(2x) // Key: x⁵ = x⁴ · x, and √(x⁴) = x² since 4 is even. // One x remains inside because its exponent (1) is odd.
Variable exponents under radicals

For √(xⁿ): divide n by 2. The quotient comes out, the remainder stays in.

√(x⁷) = x³√x    (7 ÷ 2 = 3 remainder 1, so x³ out, x¹ in)

For ∛(xⁿ): divide n by 3. Same idea with thirds.

∛(x⁸) = x²∛(x²)    (8 ÷ 3 = 2 remainder 2, so x² out, x² in)

Adding and subtracting radicals works like combining like terms — you can only combine radicals with the same index and the same radicand.

Like radicals — combining 3√5 + 7√5 = 10√5 (same radicand: combine) √12 + √75 = ? (different radicands: simplify first) √12 = 2√3 √75 = 5√3 2√3 + 5√3 = 7√3 (now they match)
Practice — Simplifying Radicals
  1. Simplify: √48
  2. Simplify: ∛(54x⁴)
  3. Simplify: √(18x³y⁶)
  4. Combine: 3√8 + √50 − √2
  5. Multiply and simplify: √6 · √15
03

Rationalizing Denominators

A simplified expression shouldn't have a radical in the denominator. "Rationalizing" means multiplying by a clever form of 1 to clear the radical from the bottom. There are two cases: a monomial denominator and a binomial denominator.

Case 1 — Monomial denominator
Rationalize: 5 / √3
Multiply top and bottom by √3: 5/√3 · √3/√3 = 5√3 / (√3 · √3) = 5√3 / 3 Answer: 5√3/3 // √3 · √3 = 3, which is rational. The denominator is now rationalized.
Conjugates

When the denominator has the form (a + √b) or (a − √b), multiplying by its conjugate eliminates the radical using the difference-of-squares pattern: (a + √b)(a − √b) = a² − b.

Case 2 — Binomial denominator
Rationalize: 4 / (3 − √5)
Multiply by the conjugate (3 + √5)/(3 + √5): 4/(3−√5) · (3+√5)/(3+√5) Denominator: (3−√5)(3+√5) = 9 − 5 = 4 Numerator: 4(3+√5) = 12 + 4√5 Result: (12 + 4√5) / 4 = 3 + √5 Answer: 3 + √5 // Notice the denominator completely cleared — no radical left.
Why bother?

Historically, rationalizing made arithmetic easier before calculators. Today it's mainly about standard form — your answer looks the same as everyone else's, which matters for checking work and for later algebraic manipulation. It also avoids having to divide by an irrational number.

Practice — Rationalizing
  1. Rationalize: 6 / √2
  2. Rationalize: √3 / √7
  3. Rationalize: 2 / (1 + √3)
  4. Rationalize: (√5 + 1) / (√5 − 1)

Complex Numbers

When we tried to take √(−1) in earlier math, we declared it "impossible." Complex numbers don't make it possible in some magical sense — they define a new object that has the property we want. That object is called i.

Definition of i

i is defined by the property: i² = −1

Equivalently: i = √(−1)

A complex number has the form a + bi, where a and b are real numbers. a is the real part, b is the imaginary part.

Powers of i — they cycle with period 4 i¹ = i i² = -1 i³ = i² · i = -i i⁴ = i² · i² = (-1)(-1) = 1 i⁵ = i⁴ · i = i ← back to start To find iⁿ: divide n by 4, use the remainder. i²³ → 23 ÷ 4 = 5 remainder 3 → i²³ = i³ = -i
Arithmetic of complex numbers

Treat i like a variable and apply the usual algebra rules — with one substitution rule: whenever i² appears, replace it with −1.

Addition and subtraction — combine real and imaginary parts separately (3 + 2i) + (1 − 5i) = (3+1) + (2−5)i = 4 − 3i (4 − i) − (2 + 3i) = (4−2) + (−1−3)i = 2 − 4i
Worked Example — Multiplication
Multiply: (2 + 3i)(1 − 4i)
FOIL: 2·1 + 2·(−4i) + 3i·1 + 3i·(−4i) = 2 − 8i + 3i − 12i² Replace i² with −1: −12i² = −12(−1) = 12 Collect real and imaginary parts: (2 + 12) + (−8 + 3)i = 14 − 5i Answer: 14 − 5i
Complex conjugates & division

The conjugate of (a + bi) is (a − bi). Their product is always real: (a + bi)(a − bi) = a² + b².

To divide complex numbers, multiply numerator and denominator by the conjugate of the denominator — same idea as rationalizing.

Worked Example — Division
Simplify: (3 + 2i) / (1 − i)
Multiply top and bottom by conjugate of denominator (1 + i): (3+2i)(1+i) 3 + 3i + 2i + 2i² ───────────── = ────────────────── (1−i)(1+i) 1 − i² Denominator: 1 − (−1) = 2 Numerator: 3 + 5i + 2(−1) = 1 + 5i Result: (1 + 5i) / 2 = 1/2 + (5/2)i Answer: 1/2 + 5i/2 // Standard form: write as a + bi with real and imaginary parts separated.
√(−n) for n > 0

Always rewrite before doing anything else: √(−9) = √(9 · −1) = √9 · √(−1) = 3i. Don't try to apply radical rules directly to negative radicands — do the separation step first.

Also: √(−4) · √(−9) ≠ √(36) = 6. The product rule for radicals doesn't apply when both radicands are negative. Do it as 2i · 3i = 6i² = −6.

Practice — Complex Arithmetic
  1. Simplify: √(−25)
  2. Simplify: i³¹
  3. Add: (5 − 3i) + (−2 + 7i)
  4. Multiply: (4 + i)(4 − i) — notice anything?
  5. Multiply: (1 + 2i)²
  6. Divide: (2 + i) / (3 + 2i)

Quadratics with Complex Solutions

Now the pieces come together. Not every quadratic has real roots — some have complex ones. The discriminant tells you which case you're in before you do any solving.

The Quadratic Formula

For ax² + bx + c = 0:

x = (−b ± √(b² − 4ac)) / 2a

The Discriminant: b² − 4ac

The expression under the radical determines the nature of the roots:

b² − 4ac > 0 → two distinct real roots
b² − 4ac = 0 → one repeated real root (touching x-axis)
b² − 4ac < 0 → two complex conjugate roots (no x-intercepts)

Connection to Week 1

The discriminant is why a degree-2 polynomial always has exactly 2 roots in the complex numbers (Fundamental Theorem of Algebra). When b² − 4ac < 0, those two roots exist — they're just not real. The graph never crosses the x-axis, but the roots are still there in ℂ.

Worked Example — Complex roots
Solve: x² − 4x + 13 = 0
Discriminant: b² − 4ac = 16 − 4(1)(13) = 16 − 52 = −36 Negative → two complex roots. Quadratic formula: x = (4 ± √(−36)) / 2 = (4 ± 6i) / 2 = 2 ± 3i Roots: x = 2 + 3i and x = 2 − 3i // Complex conjugate pair, as expected for real coefficients.
Worked Example — Completing the square
Solve by completing the square: x² + 6x + 11 = 0
Move constant: x² + 6x = −11 Complete the square: take half of 6, square it → (3)² = 9 Add to both sides: x² + 6x + 9 = −11 + 9 = −2 Factor left side: (x + 3)² = −2 Take square root: x + 3 = ±√(−2) = ±i√2 Solve: x = −3 ± i√2 // Completing the square always works; useful when the formula feels heavy.
Completing the square — the setup

The leading coefficient must be 1 before you complete the square. If you have 2x² + ..., divide everything by 2 first. Forgetting this step leads to wrong answers every time.

From roots back to polynomial — Vieta's formulas (bonus) If the roots of ax² + bx + c = 0 are r and s, then: r + s = −b/a (sum of roots) r · s = c/a (product of roots) Example: roots 2+3i and 2−3i Sum: (2+3i) + (2−3i) = 4 → −b/a = 4 → b = −4 (with a=1) Product: (2+3i)(2−3i) = 13 → c/a = 13 → c = 13 Reconstructed: x² − 4x + 13 ✓ (matches our example above)
Practice — Complex Solutions
  1. Use the discriminant to classify the roots (don't solve): 3x² − 5x + 4 = 0
  2. Solve using the quadratic formula: x² + 2x + 5 = 0
  3. Solve by completing the square: x² − 4x + 7 = 0
  4. Write a quadratic with real coefficients whose roots are 1 + 2i and 1 − 2i. What are a, b, and c?
  5. Challenge: Show that if (a + bi) is a root of a quadratic with real coefficients, then (a − bi) must also be a root. (Hint: plug a + bi into the formula and use conjugate properties.)
